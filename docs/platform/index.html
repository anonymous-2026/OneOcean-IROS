Practical guide for using the OneOcean data pipeline and environment/task stack.

Quickstart

Run data generation and the first navigation task with local variants.

python OceanEnv/Data_pipeline/run_pipeline.py --overwrite
python OceanEnv/Data_pipeline/generate_variants.py --which tiny,scene,public --overwrite
python -m oneocean_sim.cli.run_navigation_task --variant tiny --episodes 3 --output-dir runs/e1_smoke

Dataset Tiers

  • tiny: fast iteration and sanity checks
  • scene: main simulation/task construction tier
  • public: distribution-oriented, larger coverage

Task Development Flow

  1. Load variant and resolve schema contract.
  2. Select simulator adapter (S1 primary / S3 backup / S2 visual).
  3. Run task with fixed seed and export metrics.
