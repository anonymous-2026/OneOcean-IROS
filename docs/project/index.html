Problem

Existing ocean robotics benchmarks are fragmented across environment generation, dynamics assumptions, and task reproducibility. OneOcean focuses on a unified pipeline from environmental data to task execution.

Method Snapshot

We combine terrain, currents, and optional tide fields into standardized NetCDF variants and feed them into a deterministic task runner for navigation and control studies.

  • Canonical + multi-size dataset variants (`tiny`, `scene`, `public`)
  • Simulator adapters (S1 primary, S3 backup, S2 visual track)
  • Run-level metrics and reproducibility metadata
